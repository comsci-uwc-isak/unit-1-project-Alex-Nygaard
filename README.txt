This software package is a collection of bash scripts designed to assist the Car Rental Company with information flow, management and storage.
Please follow the outlined steps below to finish the installation properly.

For more information from the GitHub repository, please visit: https://github.com/comsci-uwc-isak/unit-1-project-Alex-Nygaard


********* INSTALLATION *********

To start the installation, please enter the following into the terminal:
------------

source install.sh

------------
This will install the necessary files into the right directories. Please do not move any of the files after that.


********* HELP *********

If you need help or information about the available commands, please use the following command:
------------

bash help.sh [command]

------------
Replace [command] with the name of the command you are curious about.
